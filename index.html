SVG to CSS URL Encoder

Encode SVG for use in CSS background-image, border-image, or mask

External quotes:

SVG in CSS via data URI works in all browsers only when encoded. This tool uses encodeURIComponent() so the result works everywhere.

SVG must have xmlns='http://www.w3.org/2000/svg'. If missing, it is added automatically.

Encoded SVG can be used in background, border-image, or mask.
